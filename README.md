# DataCorder

Record your data

## Quick start

```shell
python record-app.py --output-dir output --fps 12 --verbose
```

Press `r` to toggle record, press `q` to quit.

## Full usage

```shell
usage: record-app.py [-h] [--camera-device CAMERA_DEVICE] --output-dir -->
					OUTPUT_DIR [--fps FPS] [--verbose] -->
```
